# nim_stm32f3

## Description

Small embedded Nim program running on an STM32F3.

## Building

The Makefile requires the `arm-none-eabi-gcc` toolchain to be installed. To build:

```
git clone git://github.com/mwbrown/nim_stm32f3.git nim_stm32f3
cd nim_stm32f3
./setup.sh
make
```
