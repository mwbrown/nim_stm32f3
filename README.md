# nimrod_stm32f3

## Description

Small embedded Nimrod program running on an STM32F3.

## Building

The Makefile requires the `arm-none-eabi-gcc` toolchain to be installed. To build:

```
git clone git://github.com/mwbrown/nimrod_stm32f3.git nimrod_stm32f3
cd nimrod_stm32f3
./setup.sh
make
```
